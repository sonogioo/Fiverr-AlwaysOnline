# fiverr-keepalive
Apre una tab nascosta e pinnata di Fiverr, simula scroll, movimenti del mouse e clic ogni 3 minuti, cambia pagina tra inbox, dashboard e gigs ogni 15 minuti, mantiene viva la sessione con attività reali, ricrea automaticamente la tab se viene chiusa e mostra un badge verde quando è attivo e grigio quando è disattivo.
